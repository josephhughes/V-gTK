Welcome to TING's documentation!
================================


**TING** is a place holder name for a suit of tools for compiling and curating sequences and associated metadata. 
Eventually, TING will be able to handle various sequence sources such as GenBank, GISAID and local fasta files and 
offer a *simple* and *intuitive* API.
